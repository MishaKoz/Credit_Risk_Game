### How to run it locally 

1. Clone the repository:
   ```
 git clone https://github.com/MishaKoz/Credit_Risk_Game.git
 cd Credit_Risk_Game
   ```

2. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

3. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
